DSA210 Spring 2026 Project - Mehmet Alper Canıtez

## Project Proposal – Predicting Daily Productivity from Lifestyle Habits

This project aims to analyze how daily lifestyle habits affect personal productivity. The main objective is to examine whether factors such as sleep duration, study time, phone usage, and mood are related to how productive a day is. Understanding these relationships can help identify which habits contribute most to higher productivity.

The dataset will be created manually by recording daily observations. Each entry will represent one day and will include variables such as date, hours of sleep, bedtime and wake-up time, total study duration, phone screen time, number of classes attended, caffeine consumption, mood score, and a self-evaluated productivity score. The data will be stored in a spreadsheet and later exported as a CSV file for analysis. Approximately 45 days of data will be collected, resulting in a structured dataset with mostly numerical variables and a small number of categorical features if needed.

The analysis will begin with exploratory data analysis to identify patterns, trends, and correlations between variables. Hypothesis testing will be conducted to evaluate relationships such as whether longer sleep duration leads to higher productivity or whether increased phone usage negatively affects study time. In the final stage, a simple machine learning model such as linear regression or classification will be used to predict productivity based on the collected features. This project is straightforward to implement, relies on easily accessible data, and is suitable for applying fundamental data science techniques.
