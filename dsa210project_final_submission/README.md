# DSA 210 Final Project: Impact of Lifestyle Choices on Academic Productivity

## Project Overview
This repository contains the final data, code, and report for my DSA 210 Introduction to Data Science course project. The project analyzes 50 days of personal tracking data to quantify how lifestyle choices (sleep duration and smartphone screen time) impact independent study hours and daily productivity.

## Repository Contents
* `dsa210_collected_data.csv`: The complete, 50 day dataset.
* `productivity_data_features.csv`: The anonymized feature dataset used for the final analysis.
* `Milestone2_ML.ipynb` (or the name of your python notebook): The complete Python codebase containing Data Preprocessing, Exploratory Data Analysis (EDA), Hypothesis Testing, and Machine Learning.
* `report.md`: The final written report detailing the motivation, methodology, findings, and limitations.
* `requirements.txt`: The required Python libraries to run the code.
* `*.png` files: The output graphs generated by the code (Correlation Heatmap, Distributions, and ML Scatter plot).

## How to Reproduce the Analysis
To run this project locally and reproduce the findings:

1. Clone this repository to your local machine.
2. Ensure you have Python installed.
3. Install the required libraries by running the following command in your terminal:
   `pip install -r requirements.txt`
4. Open the Jupyter Notebook (`.ipynb` file) or copy the Python code into your IDE.
5. Ensure `dsa210_collected_data.csv` is in the same directory as the script.
6. Run the code from top to bottom. It will automatically generate the CSV, print the statistical results, display and save the graphs.
